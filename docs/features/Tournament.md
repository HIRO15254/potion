# トーナメント戦績管理機能

## 概要

出場したトーナメントの各種の情報を管理して、後で振り返る・各種統計を行うことができる。

## 作るデータ構造
- トーナメントテンプレート
- トーナメント結果

### サブデータ構造

- タイムスタンプ
- タグ
- プライズ

## 各データ構造詳細

### トーナメントテンプレート
- トーナメント名
- バイイン・リバイアドオン
  - 金額・点数
- ストラクチャー
- プライズストラクチャー
  - プライズデータ構造に紐づけ
  - エントリー数に応じて変化させられるように
- タグ
  - 該当データ構造に紐づけ

### トーナメント結果
- トーナメント名
- 日時
- 使用テンプレート
- 最終順位
  - 暫定・クローズ前に飛んだ等も設定できるように
- バイイン・リバイアドオン
  - 金額・点数
- プライズストラクチャー
  - プライズデータ構造に紐づけ
  - 確定したものを自動で入力できるように
- タイムスタンプ
  - トーナメント時間とデータ構造を紐づけ
- タグ
  - 該当データ構造に紐づけ

### タイムスタンプ
- 紐づける時間
- エントリー数
- リバイ・アドオン数
- 自分のスタック
- アベレージスタック
- その他メモ

### タグ
- タグ名
- 色
- 備考

### プライズ
- プライズ名
- 価値（円・ドルに対応しておく）
- 備考
